git init -> cria repositório local
git remote add origin "link do github" -> link do remetente
git add . -> junta arquivos da pasta para o staging area 
git status ->  confirma se há alguma pendência pro staging area
git commit -m "descrição do commit" -> nome da atualização do commit
git push origin main -> comitar/enviar pro github

- agora estamos na brench nova

git checkout -b dev -> cria nova branch de desenvolvimento (-b cria nova branch)
git checkout main -> muda para branch main
get checkout dev -> muda para branch dev
